对作品集有何页数 / 格式要求？

准备1套完整的App界面设计（15-20页），包含首页、核心流程页、状态页。展示你对设计规范的理解和视觉一致性把控能力。

投递前检查清单

  • 精通Figma/Skech
  • 临摹5个以上优秀App界面
  • 设计1套完整App界面
  • 了解iOS/Android设计规范
  • 学习组件化设计思维

适合哪些同学？

  • 视觉传达、数字媒体、交互设计、美术相关专业学生
